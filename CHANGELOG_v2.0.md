# OpenWrt Manager v2.0 更新日志

## 🎨 界面与体验重构 (Apple Style UI)

v2.0 版本带来了全新的用户界面设计，完全遵循 Apple Human Interface Guidelines，提供更加现代化、简洁和精致的使用体验。

### ✨ 视觉升级
- **全新设计语言**: 采用 Apple 风格的配色、字体和间距系统。
- **现代化组件**:
  - **渐变按钮**: 替换了原有的扁平按钮，采用精致的蓝色渐变风格。
  - **圆角卡片**: 界面元素采用统一的圆角和阴影设计。
  - **精致排版**: 优化了标题和文字的层级关系，信息展示更清晰。
- **暗色模式支持**: 内置深色/浅色主题系统（准备中）。

### 🚀 核心功能增强
- **多格式导出**:
  - **Excel (.xlsx)**: 支持样式导出，自动标记颜色，自动调整列宽。
  - **JSON**: 方便二次开发和数据处理。
  - **Markdown**: 方便直接粘贴到技术文档中。
  - **CSV**: 保留传统的通用格式。
  - **打印预览**: 预览打印效果（开发中）。
- **表格功能升级**:
  - **列排序**: 点击表头即可对任意列进行排序。
  - **右键菜单**: 支持复制整行、复制地址、复制凭证、重新测试和删除行。
  - **智能列宽**: 优化了表格列宽的自动适应逻辑。

### 🛠 易用性改进
- **简洁的文案**: 简化了所有功能模块的标题和引导语，降低认知负担。
- **直观的状态**: 优化了探测状态的显示方式。
- **智能交互**: 所有的按钮和输入框都增加了细腻的交互反馈。

## 📝 升级指南
请运行以下命令安装新增的依赖（用于 Excel 导出支持）：
```bash
pip install -r requirements.txt
```
